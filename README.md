# Zählen – Tier-Zählspiel

Ein Zählspiel für Kinder: 10 Haufen mit Emoji-Tieren bewegen sich über das Spielfeld. Klicke sie in der richtigen Reihenfolge an – zuerst den Haufen mit 1 Tier, dann 2, dann 3 … bis 10.

![Spielfeld](screenshot.png)

## Spielregeln

- Finde und klicke die Haufen aufsteigend von 1 bis 10
- Bei falschem Klick ertönt ein Warnton
- Am Ende wird deine Zeit angezeigt

## Starten

```
open index.html
```

Keine Installation oder Build-Schritte nötig – einfach im Browser öffnen.
